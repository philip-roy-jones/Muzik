Random Track from a Genre:
You can search for tracks within a specific genre using the Search API. For instance, you could search for tracks within the "pop" genre and then select a random result.
Use the Search API:
bash
Copy code
GET /search?q=genre:{genre_name}&type=track
Example:
For pop music, the query could be: /search?q=genre:pop&type=track.
Randomly pick a track from the returned results.

Random Track from New Releases:
You can fetch new releases and select a random song:
bash
Copy code
GET /browse/new-releases
Use this to retrieve recent songs and randomly pick one from the list.
